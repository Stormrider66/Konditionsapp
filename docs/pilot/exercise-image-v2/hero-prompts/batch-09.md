# Batch 09 Hero Frame Prompts

These prompts generate the first approved mobile hero frame for production batch 09. They use the Codex built-in image generation path unless an explicit API pipeline is chosen later.

Shared constraints for every image:

- Square 1:1, minimum 1024 x 1024.
- No exercise name, title, app UI, captions, logos, watermarks, counters, numbers, or browser chrome inside the image.
- Dark charcoal-blue sports-science gym or studio background that works in light and dark mode.
- Realistic athletic adult, full movement and equipment visible with safe mobile padding.
- Orange-red translucent muscle activation overlay on the active muscle bellies.
- Optional small uppercase Latin anatomy labels only, with thin leader lines.
- No duplicate athlete, extra limbs, warped equipment, impossible joint angles, or cropped hands, feet, bars, boxes, cables, bands, or key equipment parts.

## 1. Bar Dip

Create one premium mobile exercise demonstration image for Bar Dip.

Subject: realistic athletic adult man performing Bar Dip. Use a clean, coached, biomechanically correct bottom position on parallel dip bars: elbows bent about 90 degrees, shoulders slightly forward but controlled, chest proud, legs crossed behind or slightly tucked, full bars and full body visible.

Frame role: hero bottom frame.

Highlight these active muscles with a broad translucent orange-red anatomical overlay: triceps brachii, pectoralis major, anterior deltoid, serratus anterior, latissimus dorsi, rectus abdominis.

Avoid: ring dip, bench dip, push-up, muscle-up, shoulders collapsing, elbows flared excessively, cropped bars or feet.
