# Batch 04 Hero Frame Prompts

These prompts generate the first approved mobile hero frame for production batch 04. They use the Codex built-in image generation path unless an explicit API pipeline is chosen later.

Shared constraints for every image:

- Square 1:1, minimum 1024 x 1024.
- No exercise name, title, app UI, captions, logos, watermarks, counters, or browser chrome inside the image.
- Dark charcoal-blue sports-science gym or studio background that works in light and dark mode.
- Realistic athletic adult, full movement and equipment visible with safe mobile padding.
- Orange-red translucent muscle activation overlay on the active muscle bellies.
- Optional small uppercase Latin anatomy labels only, with thin leader lines.
- No duplicate athlete, extra limbs, warped equipment, impossible joint angles, or cropped hands, feet, plates, handles, or key machine parts.

## 1. Front Squat

Create one premium mobile exercise demonstration image for Front Squat.

Subject: realistic athletic adult woman performing Front Squat. Use a clean, coached, biomechanically correct pose: deep but controlled front squat bottom position with barbell held in front rack across the shoulders, elbows high, torso upright, heels grounded, knees tracking over toes, full bar and plates visible.

Frame role: hero bottom frame.

Highlight these active muscles with a broad translucent orange-red anatomical overlay: quadriceps femoris, gluteus maximus, adductor magnus, erector spinae, abdominal stabilizers.

Avoid: back squat bar position, crossed-arm front rack, Smith machine, elbows dropped, rounded upper back, heels lifting, knees collapsing inward, cropped plates.
