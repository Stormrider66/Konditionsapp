# Batch 05 Hero Frame Prompts

These prompts generate the first approved mobile hero frame for production batch 05. They use the Codex built-in image generation path unless an explicit API pipeline is chosen later.

Shared constraints for every image:

- Square 1:1, minimum 1024 x 1024.
- No exercise name, title, app UI, captions, logos, watermarks, counters, or browser chrome inside the image.
- Dark charcoal-blue sports-science gym or studio background that works in light and dark mode.
- Realistic athletic adult, full movement and equipment visible with safe mobile padding.
- Orange-red translucent muscle activation overlay on the active muscle bellies.
- Optional small uppercase Latin anatomy labels only, with thin leader lines.
- No duplicate athlete, extra limbs, warped equipment, impossible joint angles, or cropped hands, feet, plates, handles, rings, bars, or key equipment parts.

## 1. Lateral Lunges

Create one premium mobile exercise demonstration image for Lateral Lunges.

Subject: realistic athletic adult woman performing Lateral Lunges. Use a clean, coached, biomechanically correct pose: bottom position of a bodyweight lateral lunge with one knee deeply bent and tracking over the foot, opposite leg extended straight to the side, hips pushed back, torso tall, both feet flat.

Frame role: hero bottom frame.

Highlight these active muscles with a broad translucent orange-red anatomical overlay: quadriceps femoris, gluteus maximus, gluteus medius, adductor magnus, hamstrings, abdominal stabilizers.

Avoid: landmine barbell, forward lunge, curtsy lunge, deep Cossack stretch, knee collapse, torso folded completely forward, cropped extended foot.
