# Batch 17 Hero Frame Prompts

These prompts generate the first approved mobile hero frame for production batch 17. They use the Codex built-in image generation path unless an explicit API pipeline is chosen later.

Shared constraints for every image:

- Square 1:1, minimum 1024 x 1024.
- No exercise name, title, app UI, captions, logos, watermarks, counters, numbers, or browser chrome inside the image.
- Dark charcoal-blue sports-science gym or studio background that works in light and dark mode.
- Realistic athletic adult, full movement and equipment visible with safe mobile padding.
- Orange-red translucent muscle activation overlay on the active muscle bellies.
- Optional small uppercase Latin anatomy labels only, with thin leader lines.
- No duplicate athlete, extra limbs, warped equipment, impossible joint angles, or cropped hands, feet, bars, plates, boxes, ropes, sandbags, or key equipment parts.

## 1. Power Clean

Create one premium mobile exercise demonstration image for Power Clean.

Subject: realistic athletic adult man performing a power clean. Use a clean, coached, biomechanically correct power catch frame: barbell racked on the front shoulders, elbows high, athlete caught above parallel with hips and knees softly bent, feet flat, torso upright, full barbell and body visible.

Frame role: hero power-catch frame.

Highlight these active muscles with a broad translucent orange-red anatomical overlay: gluteus maximus, hamstrings, quadriceps femoris, erector spinae, upper trapezius, deltoids, forearm grip stabilizers, abdominal stabilizers.

Avoid: deep squat clean, deadlift setup, shoulder press, reverse curl, warped barbell, cropped plates or feet.
