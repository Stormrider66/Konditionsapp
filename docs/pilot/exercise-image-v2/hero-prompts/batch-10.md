# Batch 10 Hero Frame Prompts

These prompts generate the first approved mobile hero frame for production batch 10. They use the Codex built-in image generation path unless an explicit API pipeline is chosen later.

Shared constraints for every image:

- Square 1:1, minimum 1024 x 1024.
- No exercise name, title, app UI, captions, logos, watermarks, counters, numbers, or browser chrome inside the image.
- Dark charcoal-blue sports-science gym or studio background that works in light and dark mode.
- Realistic athletic adult, full movement and equipment visible with safe mobile padding.
- Orange-red translucent muscle activation overlay on the active muscle bellies.
- Optional small uppercase Latin anatomy labels only, with thin leader lines.
- No duplicate athlete, extra limbs, warped equipment, impossible joint angles, or cropped hands, feet, sleds, bands, kettlebells, machines, or key equipment parts.

## 1. Good Mornings

Create one premium mobile exercise demonstration image for Good Mornings.

Use a clean, coached, biomechanically correct hip-hinge bottom position: barbell on the upper back, hips pushed back, knees softly bent, torso hinged forward with neutral spine, feet grounded, full bar and plates visible.
