# Batch 08 Hero Frame Prompts

These prompts generate the first approved mobile hero frame for production batch 08. They use the Codex built-in image generation path unless an explicit API pipeline is chosen later.

Shared constraints for every image:

- Square 1:1, minimum 1024 x 1024.
- No exercise name, title, app UI, captions, logos, watermarks, counters, numbers, or browser chrome inside the image.
- Dark charcoal-blue sports-science gym or studio background that works in light and dark mode.
- Realistic athletic adult, full movement and equipment visible with safe mobile padding.
- Orange-red translucent muscle activation overlay on the active muscle bellies.
- Optional small uppercase Latin anatomy labels only, with thin leader lines.
- No duplicate athlete, extra limbs, warped equipment, impossible joint angles, or cropped hands, feet, plates, balls, ropes, bars, or key equipment parts.

## 1. Chest-to-Bar Pull-Up

Create one premium mobile exercise demonstration image for Chest-to-Bar Pull-Up.

Subject: realistic athletic adult woman performing Chest-to-Bar Pull-Up. Use a clean, coached, biomechanically plausible top position: hands gripping a pull-up bar, elbows bent and pulled down, chest clearly close to the bar, shoulders active, ribs controlled, legs slightly forward in a hollow shape, full bar and body visible.

Frame role: hero top-contact frame.

Highlight these active muscles with a broad translucent orange-red anatomical overlay: latissimus dorsi, biceps brachii, rhomboids, posterior deltoid, lower trapezius, rectus abdominis, hip flexors.

Avoid: standard chin-over-bar pull-up only, muscle-up transition, butterfly swimming, extra athlete, impossible shoulder angle, cropped bar or feet.
