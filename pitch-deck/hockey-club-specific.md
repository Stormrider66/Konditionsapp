# Star by Thomson
## Affärscase: Elithockeyklubb

## Sammanfattning

| Fakta | Värde |
|-------|-------|
| Spelare | 100 |
| Tränare | 20 |
| Tester/år | 400 |
| Träningar/vecka | 5 per spelare (500 totalt) |

---

## Problem 1: Testrapporter tar för lång tid

### Idag
- 400 tester per år
- 60 minuter per rapport
- **400 timmar per år** på rapportskrivning

### Med Star by Thomson
- Samma 400 tester
- 5 minuter per rapport
- **33 timmar per år**

### Besparing: 367 timmar = 146 800 kr/år
*(vid 400 kr/timme)*

---

## Problem 2: Ingen överblick över träning

### 500 träningspass per vecka att övervaka

| Utmaning | Konsekvens |
|----------|------------|
| Vem tränar egentligen? | Spelare kan "gömma sig" |
| Överträning | Skaderisk ökar |
| Sjukdom/trötthet | Upptäcks för sent |
| Säsongsplanering | Svår att kommunicera |

### Med Star by Thomson

| Funktion | Nytta |
|----------|-------|
| **Träningslogg** | Varje spelare loggar sina pass |
| **ACWR-övervakning** | Automatisk varning vid hög skaderisk |
| **Daglig check-in** | Sömn, energi, skador – innan träning |
| **Kalender** | Hela säsongen synlig för alla |

---

## Problem 3: Kommunikation tränare ↔ spelare

### Idag
- Program på papper/mail
- Svårt veta om spelaren förstår
- Ingen feedback-loop

### Med Star by Thomson
- Program direkt i spelarens mobil
- Spelare bekräftar genomfört pass
- Kommentarer och frågor i appen
- Tränare ser allt i dashboard
